## 🧠 Project Description

**Functional Treat (OOP Edition)** is a Python-based command-line application that demonstrates the power of Object-Oriented Programming (OOP) through a structured employee management system.

The system is built using multiple classes such as **Person, Employee, Manager, and Developer**, showcasing core OOP principles like inheritance, method overriding, and code reusability.

Users can create different types of objects, including basic persons, employees, and managers, each with their own attributes and behaviors. The system allows dynamic data entry and displays structured information through class methods.

The project highlights how real-world systems are modeled using class hierarchies. The base class (Person) provides common attributes, while derived classes extend functionality with specialized fields such as employee ID, salary, department, and programming language.

The use of inheritance ensures code reuse and scalability, while method overriding enables customized output for each class type. The menu-driven CLI interface allows smooth interaction with the system.

From an AI/ML perspective, this project builds foundational knowledge in structured data modeling and abstraction, which are essential when designing intelligent systems and working with complex datasets.

Overall, Functional Treat (OOP Edition) reflects strong programming fundamentals and a deep understanding of system design using Python.
